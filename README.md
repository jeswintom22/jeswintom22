## в–‘ featured projects

### рҹ“Ӣ AttendX вҖ” AI-Powered Attendance System
> face recognition that actually marks attendance so nobody has to

Built a full web-based attendance system using **Flask + OpenCV + face_recognition**. Students are registered with live webcam captures, and the system auto-marks attendance in real-time by recognising faces during class sessions. Includes an admin dashboard with weekly trends, a donut chart for attendance mix, eligibility warnings, Excel report export, and a classroom broadcast display. The whole thing runs on SQLite with a clean REST + JSON API layer for a Vercel frontend.

`Python` `Flask` `OpenCV` `face_recognition` `SQLite` `HTML/CSS/JS` вҶ’ [jeswintom22/attendx](https://github.com/jeswintom22/attendx)

---

### рҹ‘ҒпёҸ ClassSight вҖ” AI Classroom Assistant
> points a camera at the board, reads it, and explains it вҖ” in real-time

A real-time classroom tool built with **FastAPI + EasyOCR + Google Gemini**. Captures live video frames via WebSocket, runs OCR to extract board content, then feeds it to Gemini 1.5 Flash for instant educational explanations. Features TTL-based caching (duplicate frames resolve in <100ms), auto-capture mode with configurable intervals, session history with JSON export, and a glassmorphism UI with skeleton loaders and toast notifications.

`Python` `FastAPI` `EasyOCR` `Gemini API` `WebSockets` `Vanilla JS` вҶ’ [jeswintom22/classsight](https://github.com/jeswintom22/classsight)

---

### рҹ§  AI Skill Gap Analyzer
> upload your resume, paste a job description, get a 30-day learning roadmap

A career tool that parses PDF resumes, extracts skills using keyword matching + **sentence-transformers** for semantic similarity, compares them against job description requirements, computes a weighted readiness score, and generates a personalised AI-powered roadmap via Ollama/Mistral. Built with FastAPI, pdfplumber, scikit-learn, and a clean Jinja2 + vanilla JS frontend.

`Python` `FastAPI` `NLP` `sentence-transformers` `pdfplumber` `Ollama` вҶ’ [jeswintom22/ai-skill-gap](https://github.com/jeswintom22/ai-skill-gap)
